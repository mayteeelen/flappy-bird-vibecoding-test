This is a simple Flappy Bird clone playable in your browser. Use the spacebar or mouse click to make the bird flap and avoid the pipes. Try to get the highest score!

## How to Play
- Open `index.html` in your browser.
- Press the spacebar or click to make the bird flap.
- Avoid hitting the pipes or the ground.
- Your score increases each time you pass a pipe.

## Files
- `index.html`: Main HTML file
- `style.css`: Game styling
- `game.js`: Game logic
